  6. Embedded Finance (education, travel, health, utilities, subscriptions)

  - **MVP (March 2027):** Core banking (accounts, debit, investments)
  - **Wave 2A (Q2 2027):** Lending ecosystem (personal loans, mortgages, BNPL)

  - **Wave 2C (Q4 2027):** Monetization (wealth mgmt, premium, trading)
  - **Wave 3 (2028-2030):** Embedded finance scale (500+ partners, AED 1B+ GMV)
