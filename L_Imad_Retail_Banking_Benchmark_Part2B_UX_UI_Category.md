# CATEGORY 7: USER EXPERIENCE & USER INTERFACE (UX/UI)

## Core Retail Journeys Beyond Products

**Focus:** Non-product journeys that enable core banking activities (onboarding, account management, notifications, preferences, loyalty)

**Importance:** UX/UI is the #1 competitive differentiator in digital banking (globally, 68% of users switch banks for better mobile experience)

---

## 🎯 CATEGORY OVERVIEW

### What's Included
This category covers **core retail journeys** that are not tied to specific products:

**User Lifecycle Journeys:**
- Onboarding (new to bank, KYC, wallet setup)
- Account management (profile, settings, limits, preferences)
- Security (authentication, permissions, data privacy)
- Navigation (discovery, search, app structure)
- Notifications (alerts, preferences, frequency)
- Marketing preferences (communications, offers, opt-in/out)
- Loyalty & rewards program
- Customer support (help, chat, FAQs)
- Data privacy & consent management

**Experience Quality Metrics:**
- App rating/store rating
- Daily active users (DAU)
- Monthly active users (MAU)
- Session length
- Feature discovery rate
- Task completion time
- Error recovery time

### What's NOT Included
- Product-specific UX (cards flow, investment flow, lending flow) → covered in individual product categories
- Design systems/brand guidelines → design infrastructure
- API documentation → developer portal
