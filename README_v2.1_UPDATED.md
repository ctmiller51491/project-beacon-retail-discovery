   - Zakat calculator in MVP (non-negotiable)
   - Prayer times in MVP
   - Islamic financing explainers in all flows

   - VIP onboarding (Wave 2B)
   - Relationship manager assignment
   - Priority support (WhatsApp direct line, <30 sec)
   - Premium tier: 5-10% base = 30-40% profit

   - <5 minute mortgages (vs. market 24-48 hours)

   - <1 minute support response (vs. market 5-15 min)
